# ActivityPub Notes App — Build Steps

This docs folder contains step-by-step guides to build a mobile-first web app to read and post Ghost ActivityPub Notes.

Milestones:
- 01-auth.md — Base app + authentication (Vite, React, Tailwind, shadcn/ui) using Ghost Admin API
- 02-reader.md — Reader section (ActivityPub/Reader articles)
- 03-notes-feed.md — Notes section (ActivityPub notes feed)
- 04-post-notes-text.md — Post plain text notes
- 05-post-notes-images.md — Post images in notes

Each guide is optimized for an AI coding agent: explicit endpoints, data contracts, tasks, acceptance criteria, and security notes.
